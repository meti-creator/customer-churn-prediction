# Customer Churn Prediction

## Project Overview
This project aims to predict customer churn using machine learning techniques and provide actionable insights to improve customer retention.

## Dataset
Telco Customer Churn Dataset.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Objective
Build an end-to-end classification model to identify customers likely to churn.
